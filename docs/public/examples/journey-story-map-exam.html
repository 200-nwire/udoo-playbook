As Gal

Grade publication — not in Slice 1

As Gal

ADR-001 — timer summary

As Gal
I can see auto-aggregated scores for every auto-checkable question so that I only need to spend time on open-ended responses
EMS integration — not in Slice 1
Step 3
As Gal

As Yoav

Slice 2
Teacher's core workflow complete · manual grade transfer eliminated 4–5 weeks · Fixed estimate
Step 4
As Gal
I can set a timer on each section and lock navigation so that students cannot return to earlier sections once they move forward
As Yoav
When I lose internet during the exam, my answers are saved and my timer is paused so that a brief disconnection doesn't cost me answered questions or time
ADR-002 — exam_session
As Gal
I can enter a score for each open-ended response on a student's detail view so that I can mark all 28 submissions in one place without switching tools
As Gal
I can publish all grades in a single action so that students see their results and the system records them without any manual re-entry from my side
Step 5
As Gal
Students see a confirmation screen before leaving a locked section so that no one loses access to their answers by accidentally navigating away
As Yoav
When I reconnect after losing internet, I land on the page I was on with all my answers intact so that I can continue without having to restart

As Gal
