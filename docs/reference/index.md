# Quick Reference

The Reference section is your daily toolkit. Every template, checklist, glossary entry, and policy in this section is designed to be **copy-pasted, printed, or pulled up on a second screen** during real work.

::: tip When to use this section
You don't read the Reference section cover-to-cover. You reach for it when you're in the middle of something — creating a story, triaging a bug, preparing for a gate review, or onboarding a new team member.
:::
